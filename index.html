View project

This project is designed to manage drink orders by using Java and Java GUI. It offers several functionalities, including order taking, charge calculation, mark orders that are ready for delivery, and the creation of a waiting list and a list of completed orders. Users can modify the waiting list by removing orders and make changes to existing orders, such as updating customer details (name and address) or modifying the drinks in the order. The project supports saving and loading the lists and records, allowing for easy management and retrieval of data.
